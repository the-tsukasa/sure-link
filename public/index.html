    📱 アプリケーションのインストール

    Sure Link を Progressive Web App (PWA) としてデバイスにインストールすることで、
    ネイティブアプリケーションのような快適な体験をお楽しみいただけます。

    📱 Android デバイス（Chrome ブラウザ）

    1. ブラウザ右上のメニューアイコン 「⋮」 をタップ
    2. 「ホーム画面に追加」 を選択
    3. 「追加」をタップしてインストールを完了

    iOS デバイス（Safari ブラウザ）

    1. 画面下部の 「共有」ボタン をタップ
    2. 「ホーム画面に追加」 を選択
    3. 「追加」をタップするとホーム画面にアイコンが追加されます

    ※ アンインストールする場合は、ホーム画面のアイコンを長押しして削除してください。

    Sure Link について

    Sure Link は、リアルタイム位置情報共有とチャット機能を統合した
    次世代型ソーシャルネットワーキングサービスです。

    主な機能

    • リアルタイムメッセージング - WebSocket による即時通信
    • 位置情報共有 - Geolocation API を活用したマップ表示
    • すれ違い検出システム - 近隣ユーザーとの遭遇を通知
    • プロフィール管理 - カスタマイズ可能なユーザー情報
    • PWA 対応 - オフライン動作とプッシュ通知をサポート

    本サービスは現在ベータ版として運用されております。
    ご意見・ご要望は開発チームまでお寄せください。
